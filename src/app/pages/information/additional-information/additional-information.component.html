Porta 1024 a 65535


Por quê não liberamos as portas anteriores à 1024? São portas reservadas, conhecidas como well know ports, são usadas para serviços mais utilizados como: servidores web, compartilhamento de arquivos, etc. Um exemplo é a porta 80, ela é reservada para uso de serviços web e a porta 53, que é reservada para dns

NAT

É a tradução de endereço de rede, resumindo, é um recurso que faz a tradução de um endereço de ip privado(local) para um endereço de ip público, ele serve para que se tenha acesso à outros servidores fora da rede interna

PING

É um comando que serve para testar conectividade entre equipamentos de uma rede, resumindo, quando você envia um pacote para determinado servidor, é o ping, a resposta que este servidor te dá é o pong(sim, é do jogo ping-pong), o teste de ping serve para avaliar o tempo que o pacote demora para sair da sua máquina, chegar no servidor e voltar para sua máquina, quanto mais alto o ping, mais lento será para que possa receber o pacote que enviou anteriormente, fazendo com que ocorra atrasos na transmissão(muito utilizado em jogos)

IP

É um comando que serve para testar conectividade entre equipamentos de uma rede, resumindo, quando você envia um pacote para determinado servidor, é o ping, a resposta que este servidor te dá é o pong(sim, é do jogo ping-pong), o teste de ping serve para avaliar o tempo que o pacote demora para sair da sua máquina, chegar no servidor e voltar para sua máquina, quanto mais alto o ping, mais lento será para que possa receber o pacote que enviou anteriormente, fazendo com que ocorra atrasos na transmissão(muito utilizado em jogos)

DHCP

É um protocolo de rede que permite que os equipamentos conectados nele possam receber endereços ip automáticamente, sem a necessidade de serem setados manualmente, lembrando: Cada equipamento terá um endereço ip único na rede, se a sequência de ip's disponível acabar, possivelmente o equipamento não conseguirá se conectar

IP ESTÁTICO X IP DINÂMICO

IP ESTÁTICO: um endereço de ip que não irá se alterar mesmo que o equipamento seja reiniciado, ele será exclusivo daquele equipamento, isso claro, se este endereço ip estiver disponível na rede para que haja a conexão, do contrário, poderá acontecer um conflito de ip's, onde um equipamento já um ip único e outro equipamento está tentando obter aquele mesmo endereço ip, com isso gera o conflito


IP DINÂMICO: neste caso o endereço de ip não será setado no equipamento e sim, irá buscar um endereço ip disponível na rede para definir como seu e assim, fazer a conexão

MAC

É um endereço físico e único da placa de rede de cada equipamento, esse número é gravado diretamente na placa física e na identificação da própria placa de rede, porém ele pode ser alterado.

Diferênça entre rede 5ghz e 2.4ghz

2.4ghz: foi a primeira frequência de wi-fi criada então, todos os equipamentos que surgiram depois dela, conseguem se conectar normalmente à essa rede, mesmo um equipamento de 10 anos atrás quanto um equipamento de agora, eles possuem suporte para ela. Por ter uma frequência mais baixa a quantidade de pacotes trocados entre o roteador e o equipamento (celular, computador, etc) é menor, porém essa frequência permite que o sinal vá mais longe, atingindo áreas mais distantes. Em decorrência da frequência mais baixa, a velocidade de transmissão de dados varia entre 20mb à 50mb ou entre 50mb à 80/90mb.


5ghz: surgiu mais recentemente então não são todos os equipamentos que possuem suporte para esta frequência de rede, com isso, alguns equipamentos mais antigos não conseguem nem mesmo enxergar a rede 5ghz, isso se deve ao fato de ela possuir uma frequência de rede mais alta do que a que eles suportam. Com essa frequência mais alta, a quantidade de pacotes que podem ser trocados entre os equipamentos aumenta, fazendo com que as transferências sejam mais rápidas do que na 2.4ghz porém, pelo aumento da frequência, a distância de alcance dela é menor pois a onda que ela é enviada é mais larga, quanto mais larga a onda, menor a distância que ela alcança. A velocidade de transmissão de dados é acima de 100mb, possuindo suporte de até 1gb(1024mb) de conexão.

Causa as interferências pelo wi-fi

  • A forma como o roteador sem fio está posicionado
  • Obstáculos físicos, como paredes e pisos
  • Quaisquer outros aparelhos sem fio (monitores para bebê, controles para abrir o portão da garagem, etc.)
  • Utensílios de cozinha, tais como, micro-ondas ou geladeira
  • Outras redes wi-fi na mesma área

O que são OLTs

Um equipamento utilizado na concentração e distribuição da rede de fibra óptica fttx (fiber-to-the-x). O sinal óptico que sai da olt é direcionado ao divisor, ou splitters. São esses aparelhos que farão a divisão do sinal e da taxa de transmissão, fazendo com que ele chegue a seu destino, que é a onu ou a ont.

O que são SPLITTERs

Ele é um componente passivo da rede que serve para dividir o sinal óptico, aumentando assim a ramificação desta rede e dando-lhe mais capilaridade. assim, o splitter é capaz de distribuir (derivar) as fibras para que a olt atenda a várias onts. ou seja, ele é um elemento fundamental para aplicações ponto-multiponto.

O que é VLANs

São como ilhas que separam diferentes tipos de sub-rede, resumindo, ele separa um rede interna de outra rede interna, pensando de acordo com a conexão da onnet hoje, cada vlan que nós temos é referente à uma cidade ou à um bairro, com isso, uma rede não interfere na outra e a organização é maior.

O que são DROPs

São os cabos da fibra óptica, eles são cabos dielétricos (não conduzem eletricidade nem geram ondas eletromagnéticas) e são utilizados em redes fiber to the home (ftth), quando dizemos “drop rompido”, significa que houve algum rompimento em algum de nossos cabos de fibra, seja de algum cliente ou mesmo de alguma caixa ou olt.
